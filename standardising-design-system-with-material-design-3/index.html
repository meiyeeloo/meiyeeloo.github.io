Standardising Design System with Material Design 3

Design System

Material Design 3

Introduction

I think it is part of UI/UX designer’s responsibility to ensure the design can be built fast especially working in an IT consultancy firm.

Problem

As part of a fast-paced dev team, I realised my co-workers always used tailwind CSS as the go-to styling framework for different client’s projects.

Discover

To save time from unnecessary adjusting, I studied and introduced Material Design 3 mainly because of its ready-to-use and appealing looking components and token.

Design

Infused the components and other design elements of the framework into the design of new project.

Documentation

Followed the structure and naming convention of Material Design 3, and added explanation on customised components for better handover.
